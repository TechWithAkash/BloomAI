About the Project

Oxford 102 Flower Dataset

This application uses a Convolutional Neural Network (CNN) trained on the Oxford 102 Flower Dataset. The dataset consists of 102 flower categories commonly found in the United Kingdom. Each class consists of between 40 and 258 images.

Technology Stack

  • Backend: Flask (Python)
  • Deep Learning: TensorFlow/Keras
  • Frontend: HTML, CSS, JavaScript

Model Architecture

The model uses a custom CNN architecture optimized for flower classification. It can identify 102 different types of flowers with high accuracy.
